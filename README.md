# Zatobox for Home Assistant

Integration for all zatobox devices.

## Installation

Using HACS (Recommended)

1. Add this repo (home-assistant-zatobox)to custom
2. Search for `Zatobox` on the HACS integration page
3. Install the integration and Restart Home Assistant


## Troubleshooting

### Discovery

Make sure the zatobox devices are on the same network to be able to discover
